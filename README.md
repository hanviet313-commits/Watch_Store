*Mục tiêu của hệ thống
- Dự án xây dựng nhằm hỗ trợ quản lý sản phẩm, khách hàng và đơn hàng trên nền tảng website. 
- Hệ thống cho phép người dùng tìm kiếm, lọc sản phẩm, đặt hàng, theo dõi đơn hàng và gửi khiếu nại; đồng thời hỗ trợ nhân viên và quản trị viên quản lý sản phẩm, xử lý đơn hàng, đánh giá, hoàn tiền và theo dõi doanh thu.

*Công nghệ sử dụng
1. Ngôn ngữ lập trình
- Java: Ngôn ngữ lập trình hướng đối tượng phổ biến, sử dụng rộng rãi trong phát triển ứng dụng, đặc biệt là ứng dụng web và doanh nghiệp.
- JSP (JavaServer Pages): Công nghệ của Java cho phép nhúng mã Java vào trong các trang HTML, giúp tạo ra các trang web động.
- Servlet: Là một thành phần trong Java để xây dựng ứng dụng web, chạy trên máy chủ, xử lý yêu cầu từ phía người dùng và trả về phản hồi.
- HTML (HyperText Markup Language): Ngôn ngữ đánh dấu để xây dựng cấu trúc cho các trang web, sử dụng thẻ để định nghĩa văn bản, hình ảnh, liên kết, và các yếu tố khác trên trang web.
- CSS (Cascading Style Sheets): Ngôn ngữ mô tả dùng để định dạng giao diện trang web, như màu sắc, font chữ, khoảng cách và bố cục.
2. Hệ quản trị cơ sở dữ liệu 
- JDBC (Java Database Connectivity): Thư viện Java cho phép kết nối, truy vấn và xử lý cơ sở dữ liệu từ ứng dụng Java. JDBC là một API trung gian, giúp giao tiếp giữa ứng dụng Java và các hệ quản trị cơ sở dữ liệu khác như MySQL, Oracle, v.v.
- DAO (Data Access Object): Là một mẫu thiết kế trong lập trình, giúp cô lập phần truy xuất dữ liệu từ cơ sở dữ liệu, giúp dễ dàng quản lý các truy vấn và cải thiện tính bảo mật của hệ thống.
3. Máy chủ 
- Apache Tomcat: Một máy chủ web và servlet container mã nguồn mở cho phép triển khai và chạy các ứng dụng web Java, bao gồm JSP và Servlet.
4. Mô hình kiến trúc
- MVC (Model View Controller): Mô hình kiến trúc thiết kế phần mềm phân chia ứng dụng thành ba phần chính: Model (xử lý dữ liệu), View (hiển thị giao diện), và Controller (xử lý logic và điều khiển).

*Công cụ sử dụng
1. NetBeans IDE
NetBeans IDE (Integrated Development Environment) là một môi trường phát triển tích hợp phổ biến, hỗ trợ lập trình viên xây dựng và phát triển các ứng dụng web, phần mềm desktop, và ứng dụng di động. NetBeans hỗ trợ nhiều ngôn ngữ lập trình như Java, PHP, C++, và HTML, giúp dễ dàng phát triển và kiểm tra các chức năng của ứng dụng web. IDE này cung cấp các công cụ mạnh mẽ như trình biên dịch, trình gỡ lỗi và tính năng kiểm tra mã, giúp cải thiện chất lượng và hiệu suất lập trình.
2. MySQL Workbench
MySQL Workbench là một công cụ thiết kế cơ sở dữ liệu (database) đồ họa, được sử dụng để quản lý và thao tác với cơ sở dữ liệu MySQL. Công cụ này hỗ trợ tạo, sửa đổi và tối ưu hóa các bảng dữ liệu, truy vấn, và mô hình hóa cơ sở dữ liệu trực quan. MySQL Workbench cho phép người dùng kết nối với cơ sở dữ liệu MySQL để quản lý dữ liệu dễ dàng, đồng thời giúp phát triển, thử nghiệm các lệnh SQL và theo dõi hiệu suất của cơ sở dữ liệu.
3. Visual Paradigm
Visual Paradigm là một phần mềm thiết kế chuyên nghiệp được sử dụng rộng rãi trong lĩnh vực phát triển phần mềm và quản lý dự án. Nó cung cấp một loạt các công cụ mạnh mẽ cho việc tạo và tùy chỉnh các biểu đồ UML (Unified Modeling Language), báo cáo, và tạo mã tự động.
4. Power BI
	Power BI sẽ được sử dụng để theo dõi các KPI quan trọng, trực quan hóa các dữ liệu bán hàng, theo dõi doanh thu, số lượng đơn hàng, tỷ lệ chuyển đổi, giá trị trung bình của đơn hàng, tỷ lệ bỏ giỏ hàng, v.v.
